# Inventory Management System  
A full-stack inventory management platform built using **React**, **Spring Boot**, and **MySQL**.  
The system supports authentication, role-based access, dashboards, and complete CRUD operations for managing products, suppliers, and stock levels.

- **Inventory Management**

- **Role-Based Access Control**
  - Admin users can manage users, roles, and system configs
- **Dashboard & Reports**
  - Overview of item count, stock levels, and recent activity
- **Optimized APIs**
  - Well-structured REST APIs with pagination and filtering
- **Responsive UI**
  - Built with React + Tailwind for a clean, fast experience
